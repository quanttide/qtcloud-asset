# 原型验证

本目录作为契约驱动架构的演进实验场，通过迭代重构验证设计可行性。

## 目录结构

examples/ 包含原型脚本和验证文档。generate_product_roadmap.py 和 generate_product_roadmap.md 验证 AI 转换管道。backup_product_journal.py 和 backup_product_journal.md 验证本地文件系统适配器和归档流程。CONTRIBUTING.md 说明验证工作流和重构原则。AGENTS.md 提供文档导航和协作指南。ROADMAP.md 记录验证演进路径。

## 快速开始

运行 generate_product_roadmap.py 生成产品路线图，验证 AI 转换能力。运行 backup_product_journal.py 归档产品日志，验证文件系统操作能力。查看 ROADMAP.md 了解下一步重构方向。
