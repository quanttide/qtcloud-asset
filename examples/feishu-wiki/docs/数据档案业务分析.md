# 数据档案业务分析

## 概述

数据档案 Space（ID: 7530979408019439617）包含 **248 个节点**，是一个数据服务项目的完整资产包，涵盖了从客户需求→数据采集→处理→交付的全流程。

## 业务闭环

```
客户需求 → 数据采集 → 数据处理 → 数据模型 → 数据报告 → 交付验收
```

## 节点类型对照业务环节

| 节点类型 | 对应业务环节 | 说明 |
|---------|------------|------|
| 数据记录 | 原始数据采集 | 各类数据源的原始记录 |
| 数据处理代码 | 数据清洗/处理 | Python脚本、数据文件 |
| 数据模型 | 数据结构定义 | 专利清洗模型、专利助研模型、浙理工实践课模型 |
| 数据报告 | 交付给客户的报告 | 文本挖掘数据报告、用户文档 |
| 数据集 | 可复用的标准数据资产 | 各类爬虫项目的数据集 |
| 策划书 | 客户需求文档 | 项目策划书、数据处理方案设计 |

## 典型项目结构

### 数据报告项目（如速冻水饺电商评论）

```
数据报告/
├── 速冻水饺电商评论/
│   ├── 速冻水饺电商评论文本挖掘数据报告  ← 交付报告
│   └── 用户文档
└── 速冻水饺社媒评论/
    ├── 速冻水饺社媒评论文本挖掘数据报告
    └── 用户文档
```

### 数据集项目（如知乎数据集）

```
数据集/
└── 知乎数据集/
    ├── 汪通老师-知乎动态爬虫项目/
    │   ├── 无法爬取的用户.csv
    │   └── 知乎用户关注的人爬虫数据集.csv
    └── 汪通老师-知乎回答标签爬虫项目/
        └── 知乎回答标签爬取结果.rar
```

## 数据模型（子节点）

数据模型目录下有多个数据模型文档：

- `专利清洗数据模型/` - 2025年第11周
- `专利助研数据模型/` - 2025年第13周
- `浙理工实验课数据模型/` - 2025年第18周

这些是数据处理的规范文档，定义数据结构、清洗规则等。

## 业务价值

1. **报价依据**：积累的项目结构和复杂度可用于报价参考
2. **交付规范**：数据报告模板可复用，新项目可基于此调整
3. **经验沉淀**：数据处理脚本、清洗规则可沉淀为标准资产
4. **新人培训**：完整的项目结构帮助新人快速理解数据服务流程
