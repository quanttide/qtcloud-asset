# AGENTS.md

本目录是量潮数字资产云的演示项目，用于展示 CLI 归档功能的使用方法。

## 用途

- 演示量潮数字资产云 CLI 工具的归档功能
- 提供独立于测试数据的示例环境
- 供用户学习和实践契约驱动的归档流程

## 数据隔离原则

| 目录 | 用途 | 数据来源 |
|------|------|----------|
| `examples/demo/` | 演示示例 | 独立创建的虚构数据 |
| `assets/fixtures/` | 测试夹具 | 仅供自动化测试使用 |

## 注意事项

- 本目录的数据仅用于演示，不要用于测试
- 测试应使用 `assets/fixtures/` 目录下的数据
- 两者数据完全隔离，避免冲突
