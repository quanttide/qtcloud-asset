- **类型**: docs

- **描述**: 产品定位、数字资产边界、契约定义、工作流设计

## 概念定义

数字资产契约将隐性的工作流转化为显性的、可版本控制的、可迁移的数字资产契约。

## 核心价值

- 隐性工作流显性化
- 契约可版本控制，支持 diff、回滚、fork
- 契约不依赖特定平台，可在飞书、GitHub 等平台间迁移
