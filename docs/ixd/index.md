# 交互设计文档

## 设计原则

### 极简主义

每个界面只做一件事。用户进入界面后 3 秒内知道能做什么。隐藏不必要的选项，高级功能通过二级入口访问。

### 一致性

相同操作在不同页面表现一致。按钮位置、颜色、文案保持统一。导航结构全局一致。

### 反馈即时

用户操作后立即给出反馈。加载状态明确，成功失败都有提示。不让用户猜测系统是否在运行。

## 核心界面

### 首页仪表盘

**布局结构**

顶部是导航栏，包含产品 Logo、面包屑、用户头像。主体区域分三列展示三大核心能力的快捷入口。

左侧卡片：数字资产全景图。显示资产总数、今日变更数、成熟度分布。点击进入全景图。

中间卡片：AI 交付约束。显示规则总数、最近验证通过率、待处理告警。点击进入约束管理。

右侧卡片：数字资产定价。显示总资产估值、最近更新日、置信度。点击进入定价中心。

**底部区域**

最近变更列表，列出最近更新的 10 条资产，包含名称、类型、变更时间。点击跳转到对应资产详情。

### 资产全景图

**图谱视图**

左侧是图谱画布，占据 80% 宽度。右侧是详情面板，占据 20% 宽度，默认隐藏，点击节点后滑出。

画布支持鼠标滚轮缩放、拖拽平移。节点用不同颜色区分资产类型：文档蓝色、代码绿色、示例橙色、流程紫色。连线粗细表示关系强度，虚线表示手动关系，实线表示自动发现关系。

点击节点右侧面板滑出，显示资产名称、类型、仓库、修改时间、成熟度评分、关联列表。关联列表可点击跳转到对应节点。

**列表视图**

表格展示所有资产。列包括名称、类型、仓库、修改时间、关联数、成熟度。支持点击列头排序，支持按类型和仓库筛选。

行点击打开资产详情。行双击跳转到图谱视图并定位到该节点。

顶部有切换按钮在图谱视图和列表视图之间切换。

**顶部操作栏**

左侧面包屑：首页 > 资产全景图。右侧有刷新按钮触发手动同步，同步时显示进度条和预计剩余时间。

### 资产详情页

**左侧面板**

资产基本信息，包含名称、类型、大小、创建时间、修改时间、作者。元数据以键值对形式展示。

下方是关联资产列表，按关系类型分组展示：引用、依赖、继承、包含。每项显示名称和类型，点击跳转。

**右侧面板**

资产预览区域。Markdown 文件渲染为富文本，其他类型显示文件图标和下载按钮。

顶部有操作按钮：编辑（跳转到编辑器）、下载、删除（需确认）。

### AI 交付约束

**规则列表页**

表格展示所有规则。列包括规则名称、适用类型、状态（启用/禁用）、严重级别、最后验证时间、通过率。

操作列有按钮：编辑、启用/禁用切换、删除（需确认）。顶部有新建按钮。

支持按适用类型筛选，按通过率排序。搜索框支持按规则名模糊搜索。

**规则编辑页**

表单字段：规则名称（必填）、描述、适用资产类型（下拉多选）、验证条件列表、严重级别（单选：错误/警告）。

验证条件支持动态添加和删除。每条条件包含：类型选择（格式检查/内容检查/关联检查）、条件参数、阈值。

底部有保存和取消按钮。保存前可点击测试按钮，用示例资产验证规则是否正确。

**验证报告页**

顶部显示验证总分和通过状态（通过/不通过）。中间是各规则的验证结果列表，包含规则名、结果、失败原因、建议。

失败项高亮显示红色，警告项显示黄色。可点击展开详细信息和修复建议。

底部有重新验证按钮和导出按钮（导出为 Markdown）。

### 数字资产定价

**定价报告页**

顶部选择器支持选择资产或资产组合（树形选择器）。点击生成按钮后显示加载状态和进度条，完成后展示报告。

报告分三块：总资产估值卡片、估值明细表格、数据来源说明。

总资产估值卡片显示金额、置信度、估值日期、估值方法。估值明细表格列出各资产的名称、估值、方法、置信度。数据来源说明列出使用的数据源和数据量。

底部有导出按钮，支持 PDF 和 Markdown 格式。还有趋势分析按钮，打开折线图查看估值变化。

**价格数据管理页**

表格展示已采集的价格信号。列包括来源类型（商务报价/采购/薪酬）、金额、时间、关联资产、状态（已确认/待审查）。

顶部有导入按钮，支持 CSV 文件导入。导入后显示校验结果面板，有问题数据提示审查，可逐条修正。

支持按来源类型筛选，按时间排序。搜索框支持按资产名搜索。

### 文档工作流

**流程图视图**

垂直居中布局，背景深色。节点垂直排列，每个节点是圆角矩形，包含文档类型缩写和全称。

节点从上到下依次为：BRD（业务需求文档）、PRD（产品需求文档）、IXD（交互设计文档）、ADD（架构设计文档）、QA（测试文档）、USER（用户文档）。

节点之间用箭头连接，箭头上标注转换关系：转化、设计、驱动、验证、交付。

支持点击节点跳转到对应文档。支持悬停节点显示该文档的状态和最后更新时间。

### 系统设置

**仓库配置页**

表单配置要扫描的仓库地址，支持添加多个仓库。每个仓库有地址、分支、扫描频率（每日/每周/手动）。

下方显示最近同步记录，包含时间、状态（成功/失败）、变更资产数。

**规则模板页**

展示预定义的规则模板列表。每个模板显示名称、适用类型、包含的检查项数量。操作有预览、基于模板创建、编辑。

## 交互流程

### 查看资产全景图

用户进入系统 -> 点击资产全景图卡片 -> 默认展示图谱视图 -> 滚轮缩放查看细节 -> 拖拽平移定位 -> 点击节点右侧滑出详情 -> 点击关联资产跳转 -> 可切换到列表视图按条件筛选

### 验证 AI 交付物

智能体提交资产 -> 系统自动触发验证 -> 生成验证报告 -> 用户收到通知点击查看 -> 查看报告总分和失败项 -> 根据建议修改 -> 重新提交触发验证 -> 通过后确认接收

### 生成定价报告

用户打开定价页面 -> 选择资产范围（单选/多选/全选） -> 点击生成报告 -> 等待计算完成 -> 查看估值结果 -> 可导出 PDF/Markdown -> 可查看趋势分析

### 导入价格数据

用户进入价格数据管理 -> 点击导入 CSV -> 选择文件上传 -> 系统校验数据格式 -> 显示校验结果 -> 有问题数据标记为待审查 -> 用户逐条审查修正 -> 确认后数据入库

### 创建验证规则

用户进入规则列表 -> 点击新建 -> 填写规则名称和描述 -> 选择适用资产类型 -> 添加验证条件（选择类型、配置参数、设置阈值） -> 选择严重级别 -> 点击测试验证效果 -> 保存启用

### 同步资产变更

用户进入系统设置 -> 配置仓库地址 -> 设置扫描频率 -> 保存配置 -> 系统定时扫描 -> 发现变更生成报告 -> 用户查看变更 -> 确认关系更新 -> 知识图谱自动更新

### 浏览文档工作流

用户打开工作流页面 -> 查看垂直流程图 -> 悬停节点查看文档状态 -> 点击节点跳转到对应文档 -> 了解文档间的流转关系
